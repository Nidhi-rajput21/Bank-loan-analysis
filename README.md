# Bank-loan-analysis
End-to-end data analysis project

**Domain:** Finance / Banking Analytics  
**Tools Used:** Excel | MySQL | Power BI  

---

## Overview
This project focuses on analyzing bank loan data to understand overall loan performance, identify key financial trends, and derive actionable business insights. The workflow integrates Excel, SQL, and Power BI to create a complete data analysis and visualization pipeline.

---

## Project Workflow

### 1. Data Preparation (Excel)
- Imported and reviewed the dataset (.csv â†’ .xlsx format).  
- Performed minor data cleaning, including date formatting and consistency checks.  

### 2. Data Analysis (MySQL)
- Imported the dataset into MySQL Workbench.  
- Executed SQL queries to calculate key performance indicators (KPIs), such as:  
  - Total Loan Applications  
  - Total Funded Amount  
  - Total Amount Received  
  - Average Interest Rate  
  - Average Debt-to-Income Ratio  
- Analyzed loan quality (good vs. bad loans) and derived trends across months, states, loan terms, employee lengths, and ownership types.

### 3. Data Visualization (Power BI)
- Developed three interactive dashboards:  
  - **Summary Dashboard:** Showcased overall KPIs and high-level loan statistics.  
  - **Overview Dashboard:** Highlights monthly, regional, and categorical trends across multiple loan metrics.  
  - **Details Dashboard:** Displays detailed loan-level information for in-depth data exploration.
